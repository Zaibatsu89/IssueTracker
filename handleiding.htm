Handleiding: Flowcharts bekijken op Draw.io (app.diagrams.net)

Volg deze stappen om de flowcharts één voor één te openen en te bekijken:

  1. Ga naar de Draw.io webapplicatie: https://app.diagrams.net/
  2. Bij de vraag 'Save diagrams to', klik op Decide later.
  3. Flowchart 1 (Intake):
    1. Ga in het menu naar File → Open from → URL...
    2. Voer de volgende URL in en klik op Open:
      https://ttv-odi.nl/xs4all/jobflow/1%20Issue%20Tracker%20intake.drawio
  4. Flowchart 2 (Analyse):
    1. Ga opnieuw in het menu naar File → Open from → URL...
    2. Voer de volgende URL in en klik op Open:
      https://ttv-odi.nl/xs4all/jobflow/2%20Issue%20Tracker%20analyse.drawio
  5. Flowchart 3 (Ontwerp):
    1. Ga opnieuw in het menu naar File → Open from → URL...
    2. Voer de volgende URL in en klik op Open:
      https://ttv-odi.nl/xs4all/jobflow/3%20Issue%20Tracker%20ontwerp.drawio
  6. Flowchart 4 (Implementatie):
    1. Ga opnieuw in het menu naar File → Open from → URL...
    2. Voer de volgende URL in en klik op Open:
      https://ttv-odi.nl/xs4all/jobflow/4%20Issue%20Tracker%20implementatie.drawio
  7. Flowchart 5 (Test):
    1. Ga opnieuw in het menu naar File → Open from → URL...
    2. Voer de volgende URL in en klik op Open:
      https://ttv-odi.nl/xs4all/jobflow/5%20Issue%20Tracker%20test.drawio
  8. Flowchart 6 (Review):
    1. Ga opnieuw in het menu naar File → Open from → URL...
    2. Voer de volgende URL in en klik op Open:
      https://ttv-odi.nl/xs4all/jobflow/6%20Issue%20Tracker%20review.drawio

Elke flowchart wordt geopend in de Draw.io editor. Afhankelijk van je browserinstellingen kan dit in een nieuw tabblad zijn of het huidige tabblad vervangen.
